# Welcome to WinBind!
Hello there! Welcome to the Getting Started Guide! Here I am going to teach you to use WinBind effectively and efficiently so that you will love the experience and will use the app at its maximum!

### You would learn:
- How to read and edit the .config
- How to make and assign KeyBindings to make shortcuts easier
- How to run commands using KeyBindings
- How to simulate keys with a shortcut
- How to add alternative shortcut to shortcuts(Example: CTRL+V simulated by CTRL+SHIFT+V)
- & *much more*!
